# minds/ — mentes invocables

Reconstrucciones digitales de mentes (históricas o de referencia) que
puedes invocar para razonar usando sus patrones cognitivos reales.

**Estado inicial:** vacío. El bootstrap no instala mentes — primero
hay que tener identidad clara, luego decidir qué mentes te sirven.

**Cómo se añade una mente más tarde:**

Pídeselo al sistema en una sesión madura. Por ejemplo:

> *"Quiero poder invocar a Aristóteles para razonamiento ético."*

El sistema te ayudará a importar la mente, o a crear una nueva desde
un corpus, y la registrará aquí.

**Ejemplos de mentes que existen como referencia** (no instaladas):

- `aristotle` — las cuatro causas, razonamiento peripatético.
- `meadows` — pensamiento sistémico, leverage points.
- `cipolla` — leyes de la estupidez humana.

No son por defecto. Son opciones.
