# Email template

> Texto corto para enviar a un amigo al que le regalas el sistema.
> Adáptalo a vuestra relación — esto es un esqueleto, no un guion.

---

**Asunto:** Te he montado algo

Hola [Nombre],

Te he montado una cosa que llevo usando yo todo el día y creo que
podría servirte. No es un chatbot ni una app — es un sistema que se
queda contigo, aprende cómo trabajas, y se vuelve más útil cada
sesión que pasa.

No te mando documentación porque no se entiende leyéndola. Se entiende
usándola. Lo único que necesitas para arrancarlo:

1. Tener Node instalado (https://nodejs.org si no lo tienes ya)
2. Tener tu cuenta de Anthropic / Claude lista
3. Abrir una terminal y pegar:

```
git clone https://github.com/ignaciosatrus/personal-os-bootstrap.git
cd personal-os-bootstrap
bash install.sh
```

Te va a preguntar tu nombre y un slug corto (lo que escribirás en la
terminal para abrirlo, p.ej. `[ejemplo]`). Eso es todo.

La primera sesión la hacemos juntos — quedamos cuando puedas, te
explico el ritmo y nace contigo. No tardamos más de una hora.

Un abrazo,
Ignacio
