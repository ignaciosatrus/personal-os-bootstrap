# skills/ — skills personalizados del operador

Skills propios que crees a medida que tu sistema madura.

**Estado inicial:** vacío. Empiezas usando los skills que Claude Code
trae por defecto (learnings, update-config, loop, schedule, verify,
init, etc.) y los suyos globales en `~/.claude/skills/`.

**Cuándo crear uno aquí:**

Cuando notas que repites una secuencia de pasos en varias sesiones
distintas y quieres consolidarla. Pídelo al sistema:

> *"Esta cosa que acabo de hacer la repito mucho. Convierte el patrón
> en un skill."*

El sistema te guiará en escribir el `SKILL.md` y registrarlo.

**No copies skills "por si acaso".** Un skill que no usas es ruido.
Mejor crear el primero cuando una necesidad real aparezca.
